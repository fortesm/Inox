# Inox Trademark and Project Identity Policy

The Mozilla Public License Version 2.0 covers copyright and patent licensing for
source code. It does not grant trademark rights.

"Inox" is the name of the programming language project started by Marcelo Fortes
in 2026. The name, project identity, logos, visual marks and official branding
should not be used in a way that falsely suggests endorsement by the official
project or misleads users about the origin of a modified version.

## Allowed uses

You may use the name "Inox" to truthfully refer to:

- the Inox programming language;
- the official Inox compiler and standard library;
- compatibility with Inox;
- forks or modified versions, when clearly identified as forks or modified
  versions.

## Not allowed without permission

You may not:

- present a fork as the official Inox project;
- remove or obscure attribution to the original project;
- use the Inox name or branding in a misleading way;
- imply endorsement by Marcelo Fortes or the official Inox project when no such
  endorsement exists.

A modified distribution should use a name that clearly distinguishes it from the
official Inox project, or should clearly state that it is a modified version.
